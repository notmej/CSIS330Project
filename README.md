<h2>PLANTUML USAGE:</h2>
<ul>
  <li>step 1: download vscode + java (jdk) + configure that (to be able to run jdk (eg: code runner whatever))</li>
  <li>step 2: install plantUML extention</li>
  <li>step 3: install graphviz 
    <ul>
        <li>Windows:</li>
      <ul>
      <li> Download the .exe installer from the Graphviz website. During installation, select "Add Graphviz to the system PATH for current user".</li>
      <li> Alternatively, use Chocolatey: <code> choco install graphviz. </code> </li>
      </ul>
      <li>Mac:</li>
      <ul>
      <li> install homebrew from their page: <a href="https://brew.sh/">HomeBrew</a> </li>
      <li> write: <code>brew install graphviz</code> </li>
      </ul>
    </ul>
  </li>
  <li>step 4: open the UML file from this repo there and do whatever thats needed. :) </li>
</ul>
